# First Row Visual Prompts

Use these prompts in ChatGPT image generation for the first-row `Pourquoi Future Park ?` visuals.

Global style to keep consistent:
- Subject: Future Park Istanbul as a real-estate investment/residence concept.
- Mood: premium, analytical, sober, investor-facing.
- Palette: dark charcoal/black base, white architectural linework, restrained yellow accents, small touches of green/blue only when useful.
- Composition: clean editorial presentation image, not a marketing poster.
- Avoid: random people close-ups, readable fake UI text, watermarks, logos from Airbnb/Sahibinden/Google, busy stock-photo look, cartoon style, heavy gradients, purple/blue SaaS look.
- Output: no embedded headline text unless the prompt explicitly asks for small numeric labels. Leave enough negative space for the existing HTML text overlay.

## Implementation Notes

When the images are delivered:
- Put them under `assets-rebuilt/first-row/`.
- Keep the exact filenames listed above.
- Export PNG or WebP. PNG is safer for generated line-art; WebP is better for page weight after approval.
- Do not update `srcset` blindly. Replace `src`, `srcset`, and `sizes` together only after checking the rendered card in a CDP browser.
- Verify desktop and mobile crops before marking the visual task done.
