# gsap-core

> GSAP core animation library.

## What it does

High-performance animation library for the web. Provides tweens, timelines, custom easing, stagger, and callback control. Used for scroll-triggered animations, UI micro-interactions, page transitions, and complex motion sequences. Works with any DOM element, SVG, canvas, or JS object.
