# gsap-scrolltrigger

> GSAP ScrollTrigger plugin for scroll-driven animations.

## What it does

ScrollTrigger is a GSAP plugin that ties animations to the scroll position. Enables parallax effects, fade-in-on-scroll, pinning sections, scrub animations, toggle actions, and responsive breakpoints for scroll-linked behavior. Requires `gsap-core` as a peer.
