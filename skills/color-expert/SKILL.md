# color-expert

> Color science for production UI.

## What it does

Applies modern color science (OKLCH/OKLAB) to generate harmonious, accessible color palettes. Handles contrast ratios, light/dark mode adaptation, tonal scales, and semantic color mapping. Prevents common color mistakes like insufficient contrast, hue shifts in gradients, and inaccessible state colors.
