# frame-liquid-bg-hero

> WebGL fluid background for hero sections.

## What it does

Creates a liquid/fluid WebGL background effect for hero banners. Features metaball-like blob distortion, cursor-reactive deformation, and configurable color palettes. Uses Three.js or raw WebGL for GPU-accelerated rendering. Adds a premium, immersive feel to page hero sections without compromising performance.
